# AWS Serverless Expert (AWS Labs)

Specialized guidance for developing, testing, and deploying serverless applications.

## Key Capabilities
- **SAM/CDK Workflows:** Scaffolding and debugging serverless projects.
- **Event-Driven Design:** Implementing EventBridge and SQS patterns.
- **Durable Workflows:** Designing complex orchestrations with Step Functions.
- **Performance:** Optimizing Lambda cold starts and memory allocation.

## Best Practices
- Use the principle of least privilege for execution roles.
- Implement structured logging and distributed tracing (X-Ray).
- Leverage provisioned concurrency for latency-sensitive APIs.
