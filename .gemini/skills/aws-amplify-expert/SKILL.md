# AWS Amplify Gen 2 Expert

Use this skill to build full-stack apps with TypeScript and Amplify Gen 2.

## Key Areas
- **Auth:** Setting up Cognito with zero-config.
- **Data:** Real-time GraphQL APIs with AppSync and DynamoDB.
- **Storage:** File storage with S3.
- **Functions:** Adding custom backend logic with Lambda.

## Best Practices
- Use the code-first approach for all backend resources.
- Leverage auto-generated TypeScript clients for the frontend.
- Keep business logic in backend functions, not the UI.
