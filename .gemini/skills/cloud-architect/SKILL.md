# Cloud Architect

Guidance based on the AWS Well-Architected Framework.

## Pillars
1. **Operational Excellence:** Automate everything (IaC).
2. **Security:** Multi-layer protection (WAF, SG, IAM).
3. **Reliability:** Multi-AZ, Self-healing (ASG, ECS).
4. **Performance Efficiency:** Right-sizing, Serverless.
5. **Cost Optimization:** Pay for what you use.

## Patterns
- Microservices with ECS/Fargate.
- Serverless APIs with API Gateway/Lambda.
- Decoupled systems with SQS/SNS.
