# Bash & Linux Expert

Use this skill for shell scripting, system configuration, and command-line troubleshooting.

## Core Skills
- **Scripting:** Writing robust Bash scripts with error handling (`set -e`).
- **Piping:** Combining tools like `grep`, `awk`, `sed`, and `jq`.
- **Permissions:** Managing users, groups, and file permissions (`chmod`, `chown`).
- **Networking:** Troubleshooting with `curl`, `netstat`, `dig`, and `ssh`.

## Best Practices
- **Portability:** Use `/usr/bin/env bash` for the shebang.
- **Quoting:** Always quote variables to prevent word splitting.
- **Documentation:** Comment complex logic and provide usage help.

## Example Usage
"Use bash-linux to write a script that backups the database and uploads it to S3."
